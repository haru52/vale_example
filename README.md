# Vale Example

## 概要

[Vale](https://vale.sh/) の利用例を示すリポジトリです。

## 動作要件

|                       ツール                        | バージョン |
| --------------------------------------------------- | ---------- |
| [Vale CLI](https://vale.sh/docs/vale-cli/overview/) | >=2.18.0   |

## インストール

```sh
gh repo clone haru52/vale_example
```

## 利用方法

```sh
cd vale_example
vale .
```
